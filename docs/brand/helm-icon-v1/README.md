# Helm Icon V1

Raster brand exports for the restrained helmet-based Helm logo direction.

Source generator: Imagine Gen 2 via Codex image generation.

Selected prompt:

```text
Variation 5 of 5 for the Helm app logo and icon: restrained monochrome brand concept using an actual simplified helmet mark, slightly more technical and angular while still clean. Plain off-white background. Include exactly two assets: left, a clean wordmark reading exactly "Helm" with the helmet mark; right, a 1024x1024 iOS rounded-square app icon using the same mark. Palette: charcoal, off-white, steel gray, with no bright color. Style: flat vector-friendly geometry, crisp edges, polished iOS app icon, quiet command/control interface feel, no colorful gradients, no extra words, no devices, no shadow, no watermark.
```

Files:

- `helm-brand-sheet-v1.png`: original selected concept sheet.
- `helm-app-icon-v1-1024.png`: full-bleed app icon export for App Store source sizing.
- `helm-app-icon-v1-2048.png`: high-resolution full-bleed app icon export.
- `helm-app-icon-v1-4096.png`: maximum-resolution full-bleed app icon export.
- `helm-logo-lockup-v1-2048w.png`: wordmark lockup, 2048 px wide.
- `helm-logo-lockup-v1-4096w.png`: wordmark lockup, 4096 px wide.
- `helm-helmet-mark-v1-1024h.png`: standalone helmet mark, 1024 px tall.
- `helm-helmet-mark-v1-2048h.png`: standalone helmet mark, 2048 px tall.

Note: these are raster exports from the generated concept. The app icon exports use a square full-bleed crop so Apple platforms can apply their own icon mask. The lockup and standalone mark are intentionally kept on the light source background because the helmet uses white negative-space geometry.
